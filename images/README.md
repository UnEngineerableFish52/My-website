# Avatar Image

Place your animated avatar GIF here named `avatar.gif`.

If no GIF is present, the system will display a fallback "M" logo with the same hexagonal vessel and rotating rings effect.

## Recommended Specifications:
- Format: Animated GIF
- Dimensions: 160x160 pixels (or larger, will be scaled down)
- File size: Under 500KB for optimal performance
- Background: Transparent or matching theme colors

The avatar will:
- Drop from top after the 13-second intro
- Bounce on landing
- Follow scroll position with parallax effect
- Bounce when scroll stops
- Speed matches user scroll velocity
