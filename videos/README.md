# Background Video

Download a free sci-fi background video from Pexels:

1. Visit: https://www.pexels.com/videos/
2. Search for: "sci-fi digital" or "particles space" or "futuristic"
3. Download a video (recommended: 1920x1080, ~2-5MB)
4. Rename it to: `scifi-background.mp4`
5. Place it in this directory

Recommended videos:
- Digital Particles (abstract, cyber aesthetic)
- Futuristic Tunnel (sci-fi movement)
- Matrix Code Rain (cyberpunk style)
