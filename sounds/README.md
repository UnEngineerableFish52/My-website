## Download Instructions

To download these tracks:

1. Visit the NCS.io links above
2. Click the "Free Download" button
3. Place files in this directory with the exact names above
4. Recommended: 128kbps MP3 format for optimal size/quality balance

OR use youtube-dl:
```bash
youtube-dl -x --audio-format mp3 --audio-quality 128K https://youtu.be/p7ZsBPK656s -o deaf-kev-invincible-pt2.mp3
youtube-dl -x --audio-format mp3 --audio-quality 128K https://youtu.be/TW9d8vYrVFQ -o elektronomia-sky-high.mp3
```
